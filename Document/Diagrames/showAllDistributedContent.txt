title Show All Distributed Content
Client -> Naming: Request Remote Object
Naming -> Register: Request Remote Object
Register -> Naming: Returns ServerStub
Naming -> Client: Returns Server Stub


Client -> StubMyTubeImpl: showAllDistrubutedContent()
StubMyTubeImpl -> MyTubeImpl: showAllDistrubutedContent()
MyTubeImpl -> Server: showAllDistrubutedContent()
Server -> StubSuperServerImpl: showAllDistrubutedContent()
StubSuperServerImpl -> SuperServerImpl:  showAllDistrubutedContent()
SuperServerImpl -> SuperServer:  showAllDistrubutedContent()

loop for each Server registered in SuperServer
   SuperServer -> StubMyTubeImpl: searchAll()
   StubMyTubeImpl -> MyTubeImpl : searchAll()
   MyTubeImpl -> StubMyTubeImpl : Returns local content.
   StubMyTubeImpl -> SuperServer : Returns local content.
end

SuperServer -> SuperServerImpl: Returns all the distributed content
SuperServerImpl -> StubSuperServerImpl: Returns all the distributed content
StubSuperServerImpl -> Server: Returns all the distributed content
Server -> MyTubeImpl: Returns all the distributed content
MyTubeImpl -> StubMyTubeImpl: Returns all the distributed content
StubMyTubeImpl -> Client: Returns all the distributed content
